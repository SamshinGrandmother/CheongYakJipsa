# 🏠청약집사
[~~잘자요 아가씨~~](https://www.youtube.com/watch?v=FoO7Pmx0bE4)

## 개요 🐕
둥지를 갖고 싶지만 바쁜 당신을 위해 당신이 원하는 청약 일정을 자동으로 알려드립니다. 

북마크한 부동산에 대한 청약 일정이 새로 등록되면 문자나 메일로 알림을 보내는 서비스입니다.
